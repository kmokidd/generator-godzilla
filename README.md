# Generator Godzilla

### 简介
godzilla是帮助前端工程师快速构将项目工程的工具，基于yeoman生成

### 环境搭建
* 安装yeoman

```bash
npm install -g yo
```

* 安装godzilla模块

```bash
npm install -g generator-godzilla
```

### 使用方式：
* 创建项目的文件夹，如：godzillatest
* 命令行定位到 cd godzillatest
* 运行下面的命令
```bash
yo godzilla
```
工具将自动生成对应的项目文件以及各种工具（包含：css压缩，css合并，自动生成雪碧图，自动转换@2x图片为@1x图片，js压缩，js合并，智图压缩图片）
